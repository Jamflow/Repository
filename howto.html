<p id = "ramText">RANDOM ACCESS MEMORY (RAM): Rather compared to short-term memory, RAM is volatile (rapidly-changing)

for interacting with processed data and executing instructions. This memory is temporary and "random-accessed" for it's capability
of accessing any single byte. Byte addressability is different for systems depending on the type of storage devices connected such as hard disks and flash
drives.</p>
