HARDWARE

Hover Over Any Hardware In The Computer

HARD DRIVE: Also known as hard disks and in reference to the opposite of RAM, "Long-term memory", hard drives are mainly used to store large amounts of data in digital systems. They hold your day-to-day operating systems, executable programs, and files such as images, music, and everything dandy to you until they are removed/deleted.

MOTHERBOARD: This is the main Printed Circuit Board (PCB) that contains and allows all communication between the principal components of a system, such being the CPU, RAM, and graphics card slots. It is also reffered to as the mainframe or body of a system.

BUS: Also called the "Data Highway", those thin lines all over the motherboard are bus. They are simply physical connections that can be in the form of cables and printed circuits, allowing communication between one component to another as data travels at a speed measured in megahertz (MHz).

CENTRAL PROCESSING UNIT (CPU): Commonly referred to as the "backbone" or "brain" of a computer, this is where instructions sent from components and programs that are calculated using basic arithmetic (binary), then processed to enable the system to perform functions.

RANDOM ACCESS MEMORY (RAM): Rather compared to short-term memory, RAM is volatile (rapidly-changing) memory used by a system to store data for the CPU to process. As it is the main memory of a system, the is temporary workspace for interacting with processed data and executing instructions. This memory is temporary and "random-accessed" for it's capability of accessing any single byte. Byte addressability is different for systems depending on the type of storage devices connected such as hard disks and flash drives.

KEYBOARD: A computer keyboard is hardware set of typewriter-like keys that enable a user to enter data of text, characters, and commands into a computer or other devices. MOUSE: A mouse is a hand-operated input device used to manipulate objects and software on a computer monitor screen.

MONITOR: A computer monitor/screen is a hardware display that shows the computer interface and programs, allowing interaction with the computer through input devices typically being a keyboard and computer mouse.
